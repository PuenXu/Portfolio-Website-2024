Robot Escape Room

Course Project for Unified Robotics II - Percetion at WPI

Project Goal

To customize and program a set of three Pololu Romi robots to work together to escape from a grid-based maze that uses a variety of indicators (beacons, tags) to help the robot find the escape door. The robots need to use sensors to read information from indicators and share the information with other robots via Wi-Fi.

My Contributon

  • Developed a PD wall following algorithm to allow the robot to follow the wall of the maze using sensor data from a ultrasonic rangefinder.
  • Programmed a camera to read April tag values and communicate with the robot via I2C to share the April tag number.
  • Employed a sensor fusion algorithm to integrate readings from a gyroscope and an accelerometer to get precise pitch angle estimation.

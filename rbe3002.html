Unknown Maze SLAM and Navigation

Course Project for Unified Robotics IV - Navigation at WPI

Project Goal

To program the robot to create a map of the unknown maze with frontier exploration algorithm, and to return to the origin using A* algorithm path planning. Furthermore, the robot is to be relocated, and it needs to relocalize itself in the maze using AMCL algorithm based on the self-generated map.

My Contributon

  • Developed drive functions to allow the robot to reach any given pose upon turning to target, driving to target position, and rotating to target heading.
  • Employed board detection and dilation techniques to generate C-space of the robot with given padding.
  • Created A* search algorithms to allow the robot to find the optimal path to any point in the maze from any starting position.
